밑에는 효율성 시간초과남
```java
import java.util.*;

class Solution
{
    public int solution(int []A, int []B)
    {
        int answer = 0;

        int[] arrayA = Arrays.stream(A).sorted().toArray();
        int[] arrayB = Arrays.stream(B).sorted().toArray();

        int length = arrayA.length;
        for (int i = 0; i < length; i++) {
            int a = arrayA[i];
            int b = arrayB[length - 1 - i];
            answer += a * b;
        }

        return answer;
    }
}
```
